# MoTiles-toolkit

This toolkit contains multiple changes to the original Tiles IoT Inventor Toolkit, published by G. Francesco (https://github.com/tilestoolkit/tiles-IoT-inventor-toolkit).

Changes made to this version, known as the MoTiles MAD inventor toolkit, serve to switch the context of the Tiles toolkit, which is based in an Internet of Things (IoT) context, to a Mobile Application Development Context (MAD).
