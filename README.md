# About

This is an adapted version of the Tiles IoT inventor toolkit, published by G. Francesco (https://github.com/tilestoolkit/tiles-IoT-inventor-toolkit).

The MoTiles toolkit is an inventor toolkit which helps non-experts, such as students, generate project ideas contributing positively to the sustainable development goals in a mobile application development domain.

This repository contains the PDF files which can be downloaded in order to print the toolkit, and start playing. Pages can be printed using a regular color printer and A4 paper.

Changes made to this version, known as the MoTiles MAD Inventor Toolkit, serve to switch the context of the Tiles toolkit, which is based in an Internet of Things (IoT) context, to a Mobile Application Development Context (MAD).

# Cards

The The MoTiles toolkit is an educational toolkit which in many ways plays a lot like a card game. Through different decks, users can generate, discuss, and reflect on ideas.

- Scenario cards describe tangible problems in the domains of the 17 sustainable development goals
- Persona cards describe example users which can encounter problems in the domains of the Scenario cards.
- Technology cards provide the necessary means to design mobile applications, including common app components, purposes, internet services, human actions, sensors and feedback.
- Mission cards provide challenges to the design process that can spark creative, original ideas
- Criteria cards allow for a critical reflection on the generated idea, refining it and finalizing it for its first design iteration.

# Usage

The MoTiles toolkit is great for brainstorming sessions. Using the MoTiles playing board, users are guided through a brainstorming process. In one hour, they will explore, challenge, combine, sketch, refine and pitch to collaboratively converge towards a mobile application idea! The outcome of the workshop is a storyboard and an elevator pitch ready to be used to drive prototyping actions, collaborative discussion and reflection.
